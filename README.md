# 📚 Library Management System (Python + MySQL)

A console-based Library Management System built using Python and MySQL.  
This project helps manage library operations like adding books, issuing books, returning books, and viewing available records using Object-Oriented Programming (OOP).

---

## 🚀 Features

- Add new books
- Issue books to users
- Return books
- View all available books
- Menu-driven system for easy use
- MySQL database integration for storing data

---

## 🛠️ Tech Stack

- Python
- MySQL
- mysql-connector-python

---

## 🧠 Concepts Used

- Object-Oriented Programming (OOP)
- Classes & Objects
- Database connectivity (MySQL)
- CRUD operations
- Menu-driven program design
